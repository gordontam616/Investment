⚙️ 設定 · 真實數據來源

加 Finnhub 免費 API Key，即刻有真實報價 + 持倉新聞 + 自動通知。

① 真實報價 + 新聞（最易）
去 finnhub.io 免費註冊 → 攞 API key → 貼上面 → 撳「更新報價」。之後會自動每分鐘更新報價、每 5 分鐘抓持倉新聞，有新新聞即彈通知。

② 富途牛牛（OpenD）
富途要喺你部機行 OpenD gateway（Win/Mac/Linux），再經 Futu API（Python/JS）攞報價。純網頁駁唔到，要一個本地小程式做中轉。

③ IBKR
IBKR 行 Client Portal Gateway，登入後 API 喺 localhost:5000。同樣需要本地 gateway + 中轉先可以喺網頁攞數據。

④ K 線真實歷史
加 Twelve Data key 即用真實日線（CORS OK）。亦會自動試 Finnhub candle（需付費）同 Stooq（免 key，部分被 CORS 擋）；全部取唔到先用模擬數據。
